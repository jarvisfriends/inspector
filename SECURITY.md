# Security Policy

## Supported versions

Only the latest tagged release and the current `main` branch receive security fixes.

## Reporting a vulnerability

Please use GitHub's private vulnerability reporting for this repository rather
than opening a public issue.
You should receive an initial response within a week.

Inspector is intended for local and embedded app diagnostics. If you find code
that violates that assumption or exposes sensitive host data unexpectedly,
report it the same way.
